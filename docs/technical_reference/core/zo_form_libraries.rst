Zero-Order Model Form Libraries
===============================

WaterTap includes a library of helper methods for assembling common model forms for zero-order type models. This library includes methods for common forms of material balances for different numbers of inlets and outlets, methods for including chemical additions and different approaches for determining electricity consumption.
