How to use the Electrolyte Database (EDB)
=========================================

For information on how to install and set up the EDB, see :ref:`install-edb`.

.. note::

    Before running any of the EDB example Jupyter Notebooks, you MUST have
    MongoDB installed (see link above) locally and you need to load the 'bootstrap' data.
    See :ref:`EDB CLI How-to<how_to_use_edb_cli>` for information on
    loading the default (i.e., 'bootstrap' data).
